# Golang play ground

## Settings
* Packages
  ```
  go install github.com/go-sql-driver/mysql
  go install github.com/joho/godotenv
  ```
* Create environment variables file(.env) 

## How to
* Run
  ```
  go run xxx.go
  ```
* Build
  ```
  go build
  ```
* List packages
  ```
  go list -m all
  ```
